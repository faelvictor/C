O programa simula uma biblioteca de ROMs de SNES, onde armazena os dados de cada uma das ROM's na estrutura tabela Hash.
Contém as seguintes funcionalidades:

1 - Cadastra ROM,
	as informações de cada ROM é armazenada em uma struct, e ao preencher a struct, amazena em uma estrutura de mapa
2 - Edita ROM,
	edita cada atributo de ROM
3 - Exlcui ROM,
	exclui algum ROM
4 - Consulta ROM,
	imprime todas informaçoes da ROM
5 - Imprime Catálogo,
	imprime todas as informaçoes de todas as ROM's 

Boa parte do código foi escrito em C++
